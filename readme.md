## Description

Pix 4x sem juros - Pagaleve, gateway for WooCommerce

### Compatibility

WooCommerce version 5.0.x

### Installation

- Install and active the plugin
- `Menu` WooCommerce -> Settings -> Payments > Pagaleve

### Required

- WordPress >= 5.0
- Pagaleve account confirmed
- WooCommerce lastest version
- PHP >= 7.1
